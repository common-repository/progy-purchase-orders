Transform your online store into a multifunctional platform that allows you to create POs, manage inventory receipts, create purchase orders and more.

== Description ==

Progy Purchase Orders is a plugin that adds an extra layer of control to your online store. It allows the reception of inventories by specialized person; employees cannot change quantities without authorization. You can create vouchers (PO) and make receipt of goods quickly. The plugin automatically calculates the price paid for each item based on the item's price and delivery. An additional tool is conversion into your monetary system.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the entire `contact-form-7` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Purchase Orders' under the Woocommerce menu in your Wordpress admin panel.
